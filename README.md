# Personal Portfolio Landing Page 

## Deskripsi
Landing page personal portfolio sederhana untuk Agus Sugandi yang dibuat menggunakan HTML, Tailwind CSS, dan JavaScript. Desain responsif dengan tema gelap dan aksen warna oranye.

## Fitur
- **Desain Responsif**: Tampil optimal di desktop dan mobile
- **Navigasi Modern**: Navbar dengan menu mobile hamburger
- **Hero Section**: Judul, deskripsi profesi, dan foto profil dengan efek visual
- **Interaktivitas**: Tombol CTA dan link sosial media
- **Animasi**: Efek hover, floating animation, dan transisi smooth
- **Tema Gelap**: Background #1A1B27 dengan aksen oranye #FF6B35

## Struktur File
```
portfolio/
├── index.html          # File HTML utama
├── style.css           # Custom CSS untuk efek tambahan
├── script.js           # JavaScript untuk interaktivitas
├── img/
│   └── profile_placeholder.jpg  # Gambar placeholder profil
├── todo.md             # Daftar tugas pengembangan
└── README.md           # Dokumentasi proyek
```

## Teknologi yang Digunakan
- **HTML5**: Struktur semantik
- **Tailwind CSS**: Framework CSS utility-first
- **JavaScript**: Interaktivitas dan animasi
- **Font Awesome**: Ikon sosial media
- **Google Fonts**: Font Inter untuk tipografi modern

## Cara Penggunaan
1. Buka file `index.html` di browser web
2. Atau jalankan dengan web server lokal untuk pengalaman optimal

## Kustomisasi
- **Foto Profil**: Ganti file `img/profile_placeholder.jpg` dengan foto asli
- **Konten**: Edit teks di file `index.html`
- **Warna**: Sesuaikan palet warna di konfigurasi Tailwind
- **Link Sosial Media**: Update href pada ikon sosial media
- **CV Download**: Tambahkan link file CV yang sebenarnya

## Fitur Interaktif
- Menu mobile responsif
- Smooth scrolling untuk navigasi
- Hover effects pada elemen interaktif
- Animasi floating pada foto profil
- Alert notification untuk tombol CTA

## Browser Support
- Chrome (recommended)
- Firefox
- Safari
- Edge

## Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Catatan Pengembangan
Proyek ini dibuat sesuai dengan spesifikasi desain yang diberikan, dengan penambahan fitur interaktif dan animasi untuk meningkatkan user experience.
